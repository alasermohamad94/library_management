# إعداد Jira للمشروع

## خطوات قائد الفريق (مرة واحدة)
1) إنشاء مشروع جديد في Jira باسم **Library Management**.
2) اختيار لوحة مهام بسيطة تحتوي الأعمدة:
   - `To Do`
   - `In Progress`
   - `Done`
3) إضافة جميع أعضاء الفريق إلى المشروع.
4) إنشاء المهام الأساسية لكل مهندس حسب دوره.
5) مشاركة رابط المشروع مع الفريق.

## قواعد العمل على Jira
- كل مهندس ينشئ مهمة أو يتولى مهمة مخصصة له.
- عند بدء العمل: نقل المهمة إلى `In Progress`.
- عند الانتهاء: نقل المهمة إلى `Done`.
- إضافة رابط Pull Request داخل تعليق المهمة.
- إرفاق لقطات شاشة عند الحاجة.

## تنظيم المهام المقترح
- واجهة المستخدم: تحسين التصميم + الجدول + التوزيع.
- منطق الأعمال: إضافة/تعديل/حذف + الاستعارة والتواريخ.
- التوثيق: README + DOCS + صور Jira.
- الاختبار: TEST_PLAN + نتائج الاختبار.
