# نظام مكتبة (Library System)

## فكرة المشروع
نظام لإدارة مكتبة يتيح تتبع الكتب والأعضاء وعمليات الاستعارة والإرجاع مع البحث والتصنيف.

## أعضاء الفريق والأدوار
- محمد حمدان — قائد الفريق
- منير الخلف — واجهة المستخدم
- عاصم الرحمون — منطق الأعمال/الخدمات
- محمود هنداوي — التوثيق
- مصطفى ادلبي — الاختبار وضمان الجودة

## التقنيات المستخدم
- واجهة: HTML/CSS/JavaScript
- خادم: Node.js (Express)
- تخزين: ملف JSON بسيط (`backend/data.json`)

## طريقة التشغيل (نسخة تجريبية بالتخزين المحلي)
### المتطلبات
- متصفح حديث فقط

### تشغيل الواجهة
افتح الملف `frontend/index.html` مباشرة من المتصفح.
