# توثيق مشروع نظام المكتبة

## مراحل تنفيذ المشروع
1) جمع المتطلبات وتحليلها.
2) تصميم هيكل المشروع والواجهات.
3) تنفيذ الوظائف الأساسية.
4) الاختبار والتحسين.
5) تجهيز التوثيق والتسليم.

## تقسيم المهام
- قائد الفريق: إعداد الريبو، تنسيق العمل، مراجعة الـ PRs.
- واجهة المستخدم: صفحات العرض/البحث/التفاصيل.
- منطق الأعمال: إدارة الكتب والاستعارة والإرجاع.
- التوثيق: توثيق كامل + لقطات Jira.
- الاختبار وضمان الجودة: اختبار يدوي + خطة اختبار + التحقق قبل التسليم.

## خطة العمل
- الأسبوع 1: إعداد المشروع والهيكل الأساسي.
- الأسبوع 2: بناء الواجهة والمنطق الأساسي.
- الأسبوع 3: الاختبارات والتوثيق النهائي.

## آلية العمل باستخدام Git
1) كل عضو يعمل على فرع خاص به.
2) بعد إنهاء المهمة يتم إنشاء Pull Request.
3) المراجعة والموافقة ثم الدمج إلى `main`.
4) يجب وجود Commits واضحة باسم كل عضو.
